- HTML5 비디오 플레이어 UI 커스텀 구현 경험
- Canvas 를 활용한 frame 단위의 영상 구간 자르기 및 영역 자르기 UI 구현 경험

- Spring boot 미들웨어 구현 및 배포 경험
- 화면정의서, 디자인, 화면 개발까지 제품 구현 경험
